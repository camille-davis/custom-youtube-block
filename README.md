# Custom YouTube Block for Wordpress

Adds customizable options to YouTube embed blocks in the Gutenberg editor.

## Available Options

- Fullwidth
- Autoplay
- Loop video
- Hide controls (at the bottom)
- Hide related videos (at the end)
- Disable interaction (all mouse and keyboard interactions)

## Usage

1. Add a YouTube embed block in the Gutenberg editor
2. Select the block to open the sidebar
3. Find the "YouTube Settings" panel
4. Toggle any of the available options

## Requirements

- WordPress 5.0 or higher
- Gutenberg editor enabled
